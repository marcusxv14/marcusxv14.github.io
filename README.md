# Limpa Iguatu

Este repositório é um protótipo de site para o projeto Limpa Iguatu, uma iniciativa dedicada à gestão e monitoramento de resíduos sólidos na cidade de Iguatu. O objetivo é promover maior transparência e participação cidadã através do registro e acompanhamento de demandas relacionadas à limpeza urbana.

## Funcionalidades

- Visualização do status das demandas.
- Interface intuitiva e responsiva.
- Outras funcionalidades vão ser adicionadas no futuro

Acesse o site: [https://osmanpdo.github.io/limpaiguatu/](https://osmanpdo.github.io/limpaiguatu/)
